# Privacy

SecureeEar does not collect, sell, upload, or transmit personal data.

## Data Handling

- Camera frames are displayed locally.
- No account is created.
- No analytics SDK is included.
- No cloud endpoint is contacted by the Java application layer.
- No phone identifiers are read.
- No installed-app list is read.
- Local logs stay in the app-specific external files directory.

## Permissions

SecureeEar asks for Wi-Fi-related permissions only so Android can identify and use the local camera Wi-Fi network. It does not request contacts, SMS, call logs, microphone, broad storage, photo library, video library, or accessibility permissions.
